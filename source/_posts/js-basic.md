## 数据类型

最新ES标准定义了其中数据类型：

- 六种原生数据类型
  - Boolean
  - Null
  - Undefined
  - Number
  - String
  - Symbol (ES6中新增)
- Object

<!--more-->

### 原生数据类型

原生数据类型定义了不可修改的值，JS字符串也是不可修改的。

Boolean，值为true或者false
Null 代表空值
Undefined 代表变量没有赋值
Number JS中Number为64位双精度
String 不可修改的字符串
Symbol 定义一个唯一、不可修改的值，可以用来做对象属性的key
